점프 투 파이썬을 보고 있다.
제일 처음에 나오는 항목은 자료형이다.

현 자바 프로그래머다 보니 자바랑 많이 비교하게 된다.
언어들의 자료형은 대부분 대동소이하므로 차이가 있는 것들이나 신기한 부분 위주로 작성하겠다.

파이썬에는 소수점을 버리는 연산자가 있다.
`//` 를 사용하면 소수점을 버리게 된다.

```python
>>> 8 // 3
2
```

한가지 주의사항이 음수의 경우에는 소수점을 버리는 것이 아니라 해당 숫자의 다음 작은 정수를 나타나게 된다.

```python
>>> -7 / 5
-1.4
>>> -7 // 5
-2
```




파이썬은 브레이슬릿이 따로 없기 때문에 개행이나 들여쓰기를 잘해야 한다.
그래서 여러줄의 문자열을 표현하려면 `"""` 을 이용하면 된다.

```python
multi="""I am a boy
... You are a girl
... It's me
... """
```



파이썬은 문자열 합/곱이 가능하다

```python
>>> test = "ABCD "
>>> alpha = "is difficult"
>>>> test + alpha
'ABCD is difficult'

>>> a = "test"
>>> a * 2
'testtest'
```




파이썬은 배열값에 음수가 가능하다. 음수의 경우 뒤에서부터 찾는다.
또한 범위 지정도 가능하다.(단, 첫번째숫자이상 두번째숫자미만의 형태다)
범위 지정시 시작 인덱스를 작성하지 않으면 첫번째 부터.. 종료 인덱스를 작성하지 않으면 마지막까지 읽는다.

```python
>>> a = "I am a boy"
>>> a[0:4]
'I am'
>>> a[:6]
'I am a'
>>> a[6:]
'boy'
```

참고로 문자열은 배열로 접근한다 해도 수정할 수 없다.




문자열 관련 함수들은 아래와 같다.
count() : 문자 개수 세기
find() : 문자 위치 찾기
index() : find()와 비슷하나 해당 문자가 없을 경우 오류 발생
join() : 각각의 문자 사이에 해당 값을 삽입
upper() : 대문자로
lower() : 소문자로
lstrip() : 왼쪽 공백 삭제
rstrip() : 오른쪽 공백 삭제
replace() : 문자열 바꾸기
split() : 문자열 나누기


```python
>>> a = "test"
>>> a.count('t')
2
>>> a.find('e')
1
>>> "I am a {0}".format("boy")
>>> "I am {test} {test2=20}
```
