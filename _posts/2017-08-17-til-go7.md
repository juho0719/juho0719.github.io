import는 인터넷의 소스저장소도 사용 가능
`import "github.com/xxx`

로컬로 받아 올 수도 있음
`go get github.com/xxx`

패키지 안에서 함수, 변수, 상수는 첫 글자를 영문 소문자로 지정하면 패키지 안에서만 사용가능
첫 글자를 영문 대문자로 지정하면 외부에서 사용 가능

패키지를 컴파일하여 라이브러리로 만들려면 `go install` 사용
`go install 패키지이름`으로 하면 위치 상관없이 패키지 컴파일 가능

문서화
`godoc 패키지이름`
`godoc -http=:9090` -> 웹브라우저에서 http://localhost:9090/pkg/xxx형태로 접속가능

Print
`fmt.Printf("%+v\n", data)  // {a:1, b:2}` : 구조체 필드도 함께 표시
`fmt.Printf("%#v\n", m)     // 타입 표현` : 타입과 함께 표현
`fmt.Printf("%T\n", a)      // 데이터 타입들` : []int
