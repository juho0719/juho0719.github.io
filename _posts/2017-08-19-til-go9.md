json 사용
`import "encoding/json"`
unmarshal
```
func main() {
    doc := `
    {
        "name": "maria",
        "age": 10
    }
    `

    var data map[string]interface{} // JSON 문서의 데이터를 저장할 공간을 맵으로 선언

    json.Unmarshal([]byte(doc), &data) // doc를 바이트 슬라이스로 변환하여 넣고,
    // data의 포인터를 넣어줌

    fmt.Println(data["name"], data["age"]) // maria 10: 맵에 키를 지정하여 값을 가져옴
}
```
marshal
```
func main() {
    data := make(map[string]interface{}) // 문자열을 키로하고 모든 자료형을 저장할 수 있는 맵 생성

    data["name"] = "maria"
    data["age"] = 10

    doc, _ := json.Marshal(data) // 맵을 JSON 문서로 변환

    fmt.Println(string(doc)) // {"age":10,"name":"maria"}: 문자열로 변환하여 출력
}
```
`json.Marshal()`의 리턴값은 `[]byte`이므로 `string()`을 사용하여 출력
JSON데이터를 보기 편하도록 indent를 줄 수 있다.
`doc, _ := json.MarshalIndent(data, "", " ")`
첫번째 인자값은 데이터, 두번째는 첫 칸에 표시할 문자열(prefix), 세번째는 들여쓰기 할 문자(공백/탭 문자 사용가능)

압축
`import "compress/gzip"`
```
w := gzip.NewWriter(file) // file로 io.Writer 인터페이스를 따르는 압축 인스턴스 w 생성
defer w.Close()           // main 함수가 끝나기 직전에 압축 인스턴스를 닫음
s := "Hello, world!"
w.Write([]byte(s)) // 압축 인스턴스로 문자열을 압축하여 파일에 저장
w.Flush()          // 메모리상의 데이터를 파일에 완전히 저장
```

압축해제
```
r, err := gzip.NewReader(file) // file로 io.Reader 인터페이스를 따르는 압축 해제 인스턴스 r 생성
defer r.Close() // main 함수가 끝나기 직전에 압축 인스턴스를 닫음
b, err := ioutil.ReadAll(r) // 압축 해제 인스턴스 r을 읽으면 압축 해제된 데이터가 리턴됨
fmt.Println(string(b)) // 문자열로 변환하여 출력
```

해시 알고리즘 : MD5, SHA1, SHA256, SHA512등 데이터에서 고유한 해시 값을 추출, 이 해시값으로는 데이터를 만들어 낼 수 없으며 단방향 알고리즘이라고도 한다. 패스워드 저장시 사용한다.
대칭키 알고리즘 : AES, DES, RC4등 암호화하는 키와 복호화하는 키가 동일하다.
공개키 알고리즘 : RSA가 대표적, 암호화하는 키와 복호화하는 키가 다르다. 암호화 키는 공개 키라고 하여 외부에 공개하고, 복호화하는 키는 비밀 키(개인 키)라고 하여 외부에 노출하지 않는다.

해시 예제
```
import "crypto/sha512"

func main() {
    sha := sha512.New()          // SHA512 해시 인스턴스 생성
    sha.Write([]byte("Hello, ")) // 해시 인스턴스에 데이터 추가
    sha.Write([]byte("world!"))  // 해시 인스턴스에 데이터 추가
    h2 := sha.Sum(nil)           // 해시 인스턴스에 저장된 데이터의 SHA512 해시 값 추출
    fmt.Printf("%x\n", h2)
}
```

AES 대칭키 알고리즘
```
import "crypto/aes"

func main() {
    key := "Hello, Key 12345" // 16바이트
    s := "Hello, world! 12"   // 16바이트
    
    block, err := aes.NewCipher([]byte(key)) // AES 대칭키 암호화 블록 생성
    
    ciphertext := make([]byte, len(s))
    block.Encrypt(ciphertext, []byte(s)) // 평문을 AES 알고리즘으로 암호화
    fmt.Printf("%x\n", ciphertext)

    plaintext := make([]byte, len(s))
    block.Decrypt(plaintext, ciphertext) // AES 알고리즘으로 암호화된 데이터를 평문으로 복호화
    fmt.Println(string(plaintext))
}
```
AES는 블록 암호화 알고리즘이라 키와 데이터 크기가 일정해야 한다.
길이가 긴 데이터는 16바이트씩 잘라서 암호화 하면 되는데 이럴 경우 보안에 취약해진다. ECB라고 한다.
