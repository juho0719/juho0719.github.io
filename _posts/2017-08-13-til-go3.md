슬라이스 : 동적길이 배열
`var a []int`

make를 통해 할당 가능
```
var a []int = make([]int, 5)
var a = make([]int, 5)
a := make([]int, 5)
var b = make([]int, 5, 10)      //길이가 5이고 용량이 10
```

길이 : 인덱스로 접근할 수 있는 공간
용량 : 실제로 메모리에 할당된 공간. 용량이 가득차면 자동으로 늘어남

`len()` : 슬라이스 길이
`cap()` : 슬라이스 용량

값 추가 `a = append(a, 1, 2)`
슬라이스에 다른 슬라이스 붙이기 `a = append(a, b...)`

슬라이스는 레퍼런스 타입. 변수를 대입하면 값 복사가 아닌 주소 참조가 됨

슬라이스 복사 `copy(슬라이스A, 슬라이스B)`

부분 슬라이스 `a[0:5]`
끝 인덱스는 실제 인덱스보다 1 많아 크기가 5인 슬라이스는 [0:5]가 되어야 함


`맵[키타입]값타입` 형태
```
var a map[string]int
a := map[string]int{"Kim": 10, "Lee": 20}
a["Kim"] = 30
```

맵에 해당 키의 데이터가 있는지 확인할 땐 변수를 두개 받음
`value, a := a["Kii"]   // 0 false: 맵에 키가 없으면 두번째 리턴 값에 false가 삽입`

맵 삭제
`delete(a, "Kim")`
