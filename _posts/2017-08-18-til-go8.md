입력함수 `fmt.Scan`
`n, _ := fmt.Scan(&a, &b)   //2번째 리턴값 생략`
입력 변수 개수만큼 입력이 끝나야 다음으로 넘어감
스페이스 혹은 엔터로 입력 횟수를 체크
`n, _ := fmt.Scanf("%d,%d", &a, &b)   //형식 지정 가능`
%d를 선언하여 숫자형만 들어올 수 있고 `,`로 구분하였기 때문에 구분자는 `,`가 된다.

문자열 입출력 함수
stdout, stdin말고도 변수를 문자열로 만들거나 문자열에서 변수로 값을 가져올 수 있다.
```
fmt.Sprint("AA", "BB", "CC")   // 값을 그대로 문자열로 만듦 (AA BB CC)
fmt.Sprintln(....)             // 문자열 끝에 개행문자 삽입
fmt.Sprintf(....)              // 형식 지정 가능
Sscan도 마찬가지
```

os.OpenFile 플래그
os.O_RDONLY	읽기 전용으로 파일을 엽니다.
os.O_WRONLY	쓰기 전용으로 파일을 엽니다.
os.O_RDWR	읽기, 쓰기를 모두 할 수 있도록 파일을 엽니다.
os.O_APPEND	파일을 저장했을 때 끝부분에 내용을 추가합니다.
os.O_CREATE	파일이 없으면 파일을 새로 생성합니다. 파일이 있으면 파일을 해당 파일을 엽니다.
os.O_EXCL	파일이 없을 때만 새로 생성하며 O_CREATE와 함께 사용합니다.
os.O_SYNC	동기(Synchronous) I/O를 사용합니다.
os.O_TRUNC	파일이 있다면 파일을 연 뒤 내용을 삭제합니다.
