실수 비교시 머신 입실론보다 작거나 같은지로 확인해야 함
```
const machineEpsilon = 1e-14
fmt.Print(math.Abs(a-5.0) <= machineEpsilon)    // true이면 같은 값, 음수가 나올 수 있으므로 Abs를 사용해야함
```

복소수 : 실수부 + 허수부
허수부에는 마지막에 i붙임
`real()`는 실수부 가져오기
`imag()`는 허수부 가져오기

복소수 표현시 +기호 대신 complex()사용 가능
```
var num1 complex64 = 3 + 2.3321i
var num2 complex64 = complex(3, 2.3321);
```

byte : 16진수 / 문자 값(한글자) 
큰따옴표 사용 불가
```
var num1 byte = 0x11
var num2 byte = 'b'
var num3 byte = "a"     //에러
var num4 byte = 'aa'    //에러
```

rune : 유니코드(UTF-8)문자 코드 저장시 사용
작은 따옴표를 사용해야 하고, 문자 그대로를 저장하거나 \u(U) 를 사용하여 유니코드를 저장
```
var rune1 rune = '김'
var rune2 rune = '\ud321'
var rune3 rune = '\U000d321'
```

숫자 연산시 자료형 타입이 다르면 오류

자료형의 최대/최소 크기를 넘어가면 오버플로우/언더플로우 오류 발생

변수 크기 : `unsafe.Sizeof()`

문자열길이
한글을 UTF-8로 저장하면 한글자에 3바이트로 계산되어 길이가 나온다.
실제 길이를 구하려면 `utf8.RuneCountInString(한글))` 처럼 사용해야 함

상수
```
const a = 1
const (
    a = 1
    b = 2
    c = 3
)
const ( 
    a = iota    //0부터 순차증가
    b           //1
    c           //2
)
const ( 
    a = iota    //0부터 순차증가
    _           //_는 값 생략 시 사용 
    c           //2
)
```

변수 선언과 동시에 대입 `:=`

전역패키지 `import . "fmt"`
패키지 alias `import a "fmt"`
사용하지 않는 패키지 `import _ "fmt"`

if문
실행 후 조건 : 세미콜론으로 구분 한 후 조건식 작성
```
if testFile, err := ioutil.ReadFile("test.txt"); err == nil {
    ...
}
```

for문에 조건식만 설정하면 while과 같이 동작
```
for a < 5 {
    ...
    a += 1
}

for {   //무한루프
    ...
}
```

레이블이 있으면 break 사용시 편함
레이블과 for문 사이에 다른 로직이 있으면 에러
```
breakLable:
    for i:=0; i<j; i++ {
        for i:=0; i<3; i++ {
            break breakLable    //중첩된 반족문을 빠져나옴. AA출력
        }
    }
    
    fmt.println("AA");
```

switch문에서 해당 case실행 후 다음case문장을 실행하고 싶으면 `fallthrough`키워드 사용
단, 맨마지막case에는 사용할 수 없다.

배열의 경우 `[길이]자료형` 형태로 구성
`var a[1]int`

`...`은 초기화 할 값의 개수에 따라 자동으로 크기 설정
`a := [...]string{"Kim", "Lee", "Oh"}   //요소가 3개므로 크기는 3`

foreach문법
```
for i, value := range a {   //i에는 인덱스, value는 값, a는 배열 (생략하려면 _사용)
    ...
}
for _, value := range a {   //생략하려면 _사용
...
}
```
